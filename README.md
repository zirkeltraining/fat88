# fat88
a gargantuan 7-segment device running on smart LEDs
runs on arduino. Tested on ArduinoUno, should work on any. Prepared and pretested on NodeMCU (D1) as well.
Required Hardware:
* Strings of WS2812 / WS2818 LEDs
* Power supply.
